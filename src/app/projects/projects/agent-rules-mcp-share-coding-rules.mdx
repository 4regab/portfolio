## Overview

Agent Rules MCP is a Model Context Protocol server that allows AI coding agents to fetch and apply coding rules from any GitHub repository. It enables teams to maintain consistent coding standards across AI-assisted development workflows.

## Key Features

- **GitHub Integration**: Fetch coding rules directly from any public GitHub repository.
- **MCP Server**: Built on the Model Context Protocol for compatibility with AI coding tools.
- **Rule Sharing**: Promote consistent coding practices across teams and AI agents.
- **Easy Configuration**: Simple setup to point agents at your repository's coding standards.

## Technologies Used

- **TypeScript**: Server implementation and GitHub API integration.
- **Model Context Protocol**: For standardized communication with AI agents.
- **GitHub API**: For fetching repository rules and configurations.
