## Overview

GabayAgent is an AI-powered personal project management tool built for solo creators, students, and freelancers. It combines powerful project management features with an AI agent that handles tasks through natural conversation.

## Key Features

- **Project Management**: Kanban boards, table/timeline views, full task CRUD (states, labels, priorities, sub-tasks), calendar, dashboard, and a rich-text notes editor.
- **AI Agent**: Creates, updates, and deletes tasks, performs real-time web search (via Linkup, Tavily, and SerpAPI), generates documents (PPTX, DOCX, XLSX, PDF), and provides productivity analytics — all through natural conversation.
- **Workflows & Reminders**: Scheduled tasks via pg_cron and automated digest emails via Resend.

## Technologies Used

- **TypeScript / Next.js**: Full-stack application framework.
- **Supabase**: Database, authentication, and real-time subscriptions.
- **pg_cron**: For scheduled task workflows.
- **Resend**: For automated email notifications.
- **Linkup / Tavily / SerpAPI**: For AI-powered web search.
