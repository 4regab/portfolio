## Overview

Forest Adventure is a console-based text adventure game built as a DSA final project. It demonstrates the practical application of data structures and algorithms through an interactive forest exploration experience.

## Key Features

- **Text-Based Adventure**: Immersive storytelling through console interaction.
- **Data Structure Showcase**: Uses trees, graphs, and other structures for game world navigation.
- **Algorithm Demonstrations**: Pathfinding, sorting, and search algorithms integrated into gameplay.

## Technologies Used

- **Python**: Game logic and console interaction.
