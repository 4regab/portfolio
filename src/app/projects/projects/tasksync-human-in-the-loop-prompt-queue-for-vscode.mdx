## Overview

TaskSync is a VS Code extension that enables developers to queue prompts to Copilot with human-in-the-loop workflows. It integrates feedback loops directly into the VS Code environment, improving how developers interact with AI coding agents.

## Key Features

- **Prompt Queue**: Queue multiple prompts and tasks for AI agents, enabling batch workflows without constant context-switching.
- **Human-in-the-Loop**: Integrate feedback loops so you can review, approve, or modify AI-generated outputs before they're applied.
- **VS Code Native**: Built as a first-class VS Code extension for a seamless developer experience.
- **Open Source**: Fully open-source with 561+ stars on GitHub, backed by active community contributions.

## Technologies Used

- **TypeScript**: Core extension logic and VS Code API integration.
- **VS Code Extension API**: For deep editor integration and UI components.
- **Node.js**: Runtime for the extension backend.
