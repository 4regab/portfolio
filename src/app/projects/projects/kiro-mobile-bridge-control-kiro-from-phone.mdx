## Overview

Kiro Mobile Bridge is a lightweight mobile interface that lets you control IDE using your phone using the Chrome DevTools Protocol (CDP). It provides a seamless way to interact with AI agents from any mobile device.

## Key Features

- **Mobile-First Interface**: Clean, responsive UI designed for phone screens.
- **CDP Integration**: Uses Chrome DevTools Protocol for reliable, low-latency communication.
- **Remote Control**: Interact with Kiro from anywhere on any device with a browser.
- **Lightweight**: Minimal footprint — no heavy dependencies or complex setup required.

## Technologies Used

- **JavaScript**: Core application logic.
- **Chrome DevTools Protocol**: For browser-level communication and control.
- **Node.js**: Server runtime.
